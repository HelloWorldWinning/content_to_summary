**3. 四种AI写图的全新范式演示**

**范式一：使用Cloud Code直接生成Excalidraw图表 (从零到一的自由创作)**
*   **目标：** 将全新、复杂的思想快速视觉化。
*   **工具：** Cloud Code (可与本地文件系统交互)。
*   **方法：**
    1.  建立独立工作目录（规范管理，避免AI误操作）。
    2.  通过逆向工程Excalidraw文件，生成“蓝图Prompt”。
    3.  利用“蓝图Prompt”指导Cloud Code，根据提供的文章内容（如MAPS思维框架）绘制Excalidraw图表。
*   **优势：** 实现思想的快速可视化，且可无限复用。

**范式二：使用GEMINI CLI生成Obsidian Canvas画板 (对现有知识的结构化布局)**

*   **工具：** GEMINI CLI (生成结构), Cloud Code (美化)。
*   **方法：**
    1.  同样生成针对Obsidian Canvas的“蓝图Prompt”。
    2.  让GEMINI CLI根据文章（如“橘子树模型”）生成Canvas文件。
    3.  利用Cloud Code对GEMINI生成的Canvas进行美化（例如调整布局、颜色、表情符号等）。
    4.  进一步演示让AI从笔记库中查找相关笔记并插入到Canvas中，建立引用链接。
*   **优势：** 强大的结构化布局能力，可有机连接现有知识，发现新洞察；演示AI间协同工作的灵活性。

**范式三：使用Cloud Code生成具有交互效果的HTML演示页面 (公开分享)**
*   **目标：** 将想法公开分享，发布为可交互的网页。
*   **工具：** Cloud Code (为代码而生)。
*   **方法：**
    1.  利用课程讲义展示页面的“蓝图Prompt”。
    2.  指导Cloud Code根据文章内容（如“橘子树模型”）创建STML（HTML）页面。
*   **优势：** AI成为高效的前端开发工具，轻松将结构化思想转化为可即刻发布的网页。

**范式四：使用Ganwe (Gamma) 一键生成PPT (传统商务演示)**
*   **目标：** 快速应对同步、线下的传统商业演示场景。
*   **工具：** Ganwe (Gamma.app)。
*   **方法：**
    1.  直接将文本内容（如MAPS思维框架）粘贴到Ganwe的AI创建功能。
    2.  选择“演示文稿”类型，配置文本处理方式（压缩/保留原样）、语言、图片来源等。
    3.  选择主题并一键生成多页PPT，自动配图。
*   **优势：** 极其高效、便捷，适用于高频次的演示需求。
