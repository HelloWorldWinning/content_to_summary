**核心要点：**

*   **Stalk：量化交易好帮手**：这是一个基于 GitHub 的开源项目，旨在帮助用户进行股票量化分析和交易。它具有选股、K线分析、回测交易和策略制定等功能。
*   **两种安装方式**：
    *   **Docker 安装 (推荐初学者)**： 简单易用，视频教程也以 Docker 版为主。需要启动两个 Docker 容器：一个是数据库服务，一个是 Stalk 本身服务。
    *   **Windows 系统安装**：主要区别在于自动交易模块，该模块只能在 Windows 下使用。
*   **功能概述**：
    *   **选股功能**：类似于在线选股工具，可以根据各种指标进行筛选，并保存筛选结果。
    *   **股票基本数据**：提供股票的资金流向、龙虎榜、大宗交易等数据。但文中作者发现每日数据中缺少交易量数据（可能为版本bug）。
    *   **K线图分析**：K线图上会标注各种形态，并提供形态说明链接。K线图组件操作稍有不便。形态数值，正数代表买入信号，负数代表卖出信号
    *   **策略选股**：内置一些策略（如放量上涨），并支持自定义策略（需要修改代码和数据库）。
    *   **自动交易 (Windows限定)**：  作者未测试。
    *   **数据同步**：通过定时任务同步数据，需要等待同步任务启动后才会有数据。
*   **调试与维护**：
    *   **数据库查看**：可以通过 Docker 进入数据库容器，进行数据查询和导出。
    *   **系统日志**：可以通过进入 Stalk 服务容器查看执行任务和外部日志。
    *   **同步时间**：可以通过查看 Dockerfile 了解定时任务的执行频率。
*   **技术架构**：
    *   **分析部分**：Python 编写。
    *   **外部服务**：基于 Golang 的 Headless Chrome (类似于一个无头的浏览器)实现。
    *   **前端界面**：使用 SpreadsheetJS（在线表格组件），K线图使用 Bokeh 表格组件，自动交易模块使用同花顺 EZ Trader。
